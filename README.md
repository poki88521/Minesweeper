# 扫雷(Minesweeper) V1.0

## 0.免责声明
此项目仅为学习使用，对可能造成的电脑问题致歉（不是故意的）<br>

## 1.简介
开发语言：java（JDK 25）<br>
基础功能：简单（10 * 10）， 普通（15 * 15）， 困难（25 * 25）以及自定义难度的扫雷游戏<br>
程序中还有其他细节，会有相关提示<br>
~~有继续更新的计划但不一定继续更新~~<br>

## 2.安装
（1）必须安装JDK25！！（版本设置不当致歉）（Oracle官方的下载链接在[这里](https://www.oracle.com/java/technologies/downloads/)）<br>
（2）点击setup.exe安装在你想安装的地方<br>
（3）点击Minesweeper.exe或桌面快捷方式运行即可<br>
